# API 명세서
- 할 일 관리 API
  - 할 일 생성 : POST   /todo
  - 할 일 조회 : GET    /todo/list
  - 할 일 수정 : PATCH  /todo/{todo_id}
  - 할 일 삭제 : DELETE /todo/{todo_id}
  - 할 일 체크 : POST   /todo/{todo_id}/check
  - 할 일 체크 해제 : POST /todo/{todo_id}/uncheck

- 유저 관리 API
  - 회원 가입
  - 로그인

- 친구 API
  - 친구 추가 요청/수락/거절
  - 친구 조회/삭제
  - 친구의 todo list 접근
